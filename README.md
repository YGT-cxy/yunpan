# yunpan
介绍：类似云盘的一个开源项目--只有前端部分的文件。后续会不断更新。
# 部署方式
 1. 方式一：下载dist目录的文件(已编译好的)，直接使用apache访问即可。
 2. 方式二：下载整个项目目录，项目需要node才能进行编译，所以需要下载node，然后按照项目需要的依赖包。
	> 安装依赖包的命令： `npm install`
	
    > 启用node服务器：`npm run server`
    
    > 编译src目录文件到dist目录：`npm run start`
